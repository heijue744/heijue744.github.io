问：喝绿茶要注意什么

答：最好淡些，下午三点后不能喝。

问：便秘吃香蕉还是喝水？

答：香蕉本身膳食纤维含量就不高，而目香蕉会分泌大量酸，抑制肠道蠕动，反而不好。

问：吃啥容易爆痘？

答：辛辣食物可能导致体内炎症反应，引发或加重痤疮（粉刺）等皮肤问题。

问：饭菜放冰箱里是趁热还是趁凉？

答：将热食物放入冰箱有助于快速降温，避免细菌滋生。为了其他食物的温度，建议稍微冷却再放

问：手机会消耗气血吗？

答：手机使用过度可能导致眼睛疲劳，但不会直接消耗气血。

问：鸡蛋怎么吃到百分百的营养？

答：白水煮着吃。

问：睡前可以吃点小点心吗？

答：可以吃，但是只能少量吃。
